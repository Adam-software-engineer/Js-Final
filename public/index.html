Home Page / - Create a home page that displays the name of the food truck and a brief description. This page will also display the events where the food truck will be located. When the user clicks on an event, they should be shown more information about the event, including the location, dates, and hours.

Menu Page /menu - Create a menu page that displays the items available for purchase. Each item should include a name, description, and price.

Contact Page /contact - Create a contact page that displays the contact information for the food truck. Include a (non-working) contact form.

Admin Page /admin - Create an admin page that allows the food truck owner to manage the menu and events. The admin page should include forms to add, update, and delete menu items and events.
